\section{Objetivos de la iteración}
El objetivo que se ha diseñado para esta primera iteración es el prototipado de las funcionalidades básicas, como registro e identificación, así como subir una publicación y seleccionar un viaje.

Esta iteración generará prototipos visuales para que el cliente pueda ver cómo serán las funcionalidades básicas de la aplicación.

\section{Desviaciones previstas}

Durante el sprint se han realizado las siguientes modificaciones en las historias de usuario:

\begin{itemize}
\item \textbf{HU-\ref{hu:subir}}. Se añade el campo ``Título'' y ``Enlace''. 
\end{itemize}

Añadimos las siguientes historias de usuario a la lista del producto:

\begin{itemize}
\item \hu{hu:buscar}. Un usuario quiere buscar una publicación a partir de un texto.
\item \hu{hu:ver_datos}. Un usuario quiere ver su información personal.
\item \hu{hu:modificar_datos}. Un usuario quiere modificar sus datos personales.
\item \hu{hu:marcar_realizado}. Un usuario quiere indicar que ha realizado una actividad.
\item \hu{hu:cerrar_sesion}. Un usuario desea cerrar la sesión en la aplicación.
\item \hu{hu:ver_publicadas}. Un usuario quiere ver sus propias publicaciones.
\item \hu{hu:eliminar_deseo}. Un usuario quiere eliminar una actividad de su lista de deseos.
\item \hu{hu:eliminar_voy_a_realizar}. Un usuario desea eliminar una actividad de su lista de actividades a realizar.
\item \hu{hu:recuperar_contraseña}. Un usuario ha olvidado su contraseña y desea poder recuperarla.
\item \hu{hu:saber_mas}. Un usuario desea conocer más información sobre una publicación en particular.
\item \hu{hu:sobre_nosotros}. Un usuario quiere saber más informacón sobre quién ha desarrollado la aplicación.
\item \hu{hu:ayuda}. Un usuario desea ver una guía sobre el uso de la aplicación.
\item \hu{hu:eliminar_publicacion}. Un usuario desea eliminar una publicación que ha subido. 
\item \hu{hu:modificar_publicacion}. Un usuario desea modificar una publicación que ha subido
\item \hu{hu:ver_realizar}. Un usuario quiere ver la lista de publicaciones a realizar.
\item \hu{hu:ver_realizadas}.  Un usuario desea ver su lista de publicaciones ya realizadas. 
\end{itemize}
